## 🚀 快速开始

### 本地预览

1. 克隆仓库：
```bash
git clone https://github.com/nexori-studio/nexori-studio.github.io.git
cd nexori-studio.github.io
```

2. 使用任意静态服务器预览：
```bash
# Python
python3 -m http.server 8080

# Node.js
npx serve

# PHP
php -S localhost:8080
```

3. 访问 `http://localhost:8080`
